Bug Tracker

Bug Tracker

'Bug Tracker' is a software defect tracking system. 'Bug Tracker' is built with ASP.NET, MVC, C# and SQL. A role based security system was implemented to create permission based roles. The roles of Admin, Project Manager, Developer and Submitter restrict what the user can see and do. The software allows for users to submit tickets and track those tickets throughout their cycle from open to complete.

The system was built with C# and .Net in 4 weeks. The front end was developed with HTML, JavaScript and Bootstrap and CSS. The back-end utilizes MVC and C#. The database was built with PostgreSQL and hosted on Heroku. Please visit the site and try it out.

  • Project Type:

    Web Application

  • Technologies:

    C#, SQL, ASP.NET Core MVC, JS, CSS, Bootstrap, HTML5
