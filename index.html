Bug Tracker

Bug Tracker

'Bug Tracker' is a software defect tracking system. 'Bug Tracker' is built with ASP.NET, MVC, C# and SQL. A role based security system was implemented to create permission based roles. The roles of Admin , Project Manager, Developer and Submitter restrict what the user can see and do. The software allows for users to submit tickets and those tickets are tracked through status from open to complete.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies MVC and C#. The database was built with SQL Server and hosted on Azure. Please visit the site and try it out.

  • Technologies:
